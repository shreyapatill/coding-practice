# Competitive Coding Solutions

Solutions for **LeetCode** and **Work@Tech** problems.

## Structure

`solutions/<platform>/<difficulty>/<solution-file>`

Example: `solutions/leetcode/easy/leetcode-1-two-sum-easy.py`

## Usage

1. **Clone** the repository.
2. **Navigate** to the platform and difficulty folder.
3. **Run** the solution file.
